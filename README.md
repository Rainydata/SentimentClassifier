# SentimentClassifier 🚀

Text sentiment classifier using Machine Learning with Python. Train a model to detect whether a text has a **positive**, **negative**, sentiment, ideal for sentiment analysis, reviews, or comments.

---

## 🧠 Technologies used

- Python 3.x
- Scikit-learn
- NLTK / spaCy 
- Pandas
- Joblib
